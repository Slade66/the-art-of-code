- **将代码划分为“段落章节”：**
	- **大块分隔：**
		- 使用 `// ========================================` 搭配大写描述性标题标记主要章节。
			- ```go
			  // ============================================================
			  // 核心数据结构定义 (CORE DATA STRUCTURES)
			  // ============================================================
			  type User struct {
			      ID   int
			      Name string
			  }
			  
			  // ============================================================
			  // 全局常量 (GLOBAL CONSTANTS)
			  // ============================================================
			  const MaxRetries = 5
			  ```
	- **空行：**
		- 在逻辑段落之间应始终保留一行空行，空行本身即可清晰地划分不同的逻辑块。
	- **小步骤分隔：**
		- 在函数内部，分隔不同逻辑步骤时，使用 `// x. 代码块的描述` 并配合空行。
			- ```go
			  func processRequest(req *Request) (*Response, error) {
			      // 1. 输入验证
			      if req == nil {
			          return nil, errors.New("request is nil")
			      }
			  
			      // --- 2. 数据库查询 ---
			      user, err := db.GetUser(req.UserID)
			      if err != nil {
			          return nil, fmt.Errorf("failed to get user: %w", err)
			      }
			  
			      // 3. 业务逻辑处理
			      response := buildResponse(user, req.Data)
			  
			      return response, nil
			  }
			  ```
