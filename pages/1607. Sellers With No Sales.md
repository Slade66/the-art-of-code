- ## Description
	- **题目链接：**https://leetcode.cn/problems/sellers-with-no-sales/
	- **题目截图：**
		- ![1607. Sellers With No Sales.png](../assets/1607._Sellers_With_No_Sales_1754735628247_0.png)
	- **题目意思：**
		- 找出 2020 年无销售记录的卖家，并返回其名字。
- ## Solution A
	- ### Algorithm
		- 筛选出 2020 年的销售记录。
		  logseq.order-list-type:: number
		- 从这些记录中提取唯一的 `seller_id`，得到 2020 年有销售的卖家集合 `sellers_with_sales`。
		  logseq.order-list-type:: number
		- 将 `Seller` 表和 `sellers_with_sales` 表左连接。
		  logseq.order-list-type:: number
		- 找出连接结果中 `seller_id` 为 `NULL` 的记录，即 2020 年没有销售的卖家。
		  logseq.order-list-type:: number
		- 按 `seller_name` 升序排序，得到最终结果。
		  logseq.order-list-type:: number
	- ### Code
		- ```mysql
		  WITH sellers_with_sales AS (
		      SELECT DISTINCT seller_id FROM Orders WHERE YEAR(sale_date) = 2020
		  )
		  
		  SELECT seller_name FROM Seller AS a LEFT JOIN sellers_with_sales AS b ON a.seller_id = b.seller_id WHERE b.seller_id IS NULL ORDER BY seller_name ASC;
		  ```
