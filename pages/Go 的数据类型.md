`rune`
heading:: true
	- `rune` 是 `int32` 的别名，用于表示一个 Unicode 码点（即某个字符对应的数字编号，而不是字符本身）。
	- `rune` 字面量使用单引号包围。

- 类型转换
  heading:: true
	- 类型转换用于将一个值从一种类型转换为另一种类型。因为 Go 不会自动进行隐式类型转换，所以需要手动指定转换方式。
	- 不同类型的变量不允许进行数学或比较运算。
	- ```go
	  语法：目标类型(待转换的值)
	  示例：int(1.4) // 结果为 1
	  ```
	- 注意：
		- 浮点数转换为整数时，会直接舍去小数部分，不进行四舍五入。
		- 数值与字符串之间的转换通常需要使用 `strconv` 包。
		- 大整数变量转换为小整数类型时，会截断为小类型所能表示的位数范围，超出部分将被舍弃，但不会报错。
		- 大浮点数转换为小浮点数时，会发生精度丢失，但不会报错。
