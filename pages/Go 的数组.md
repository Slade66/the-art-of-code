- 数组是一种用于存储相同类型元素的数据结构。
  title:: Go 的数组
- 数组的长度在创建时就确定，固定不变。
- 数组中的每个值称为元素，创建时都会被初始化为该类型的零值。
- 数组类型由元素类型和长度共同决定，数组长度是类型的一部分，因此 `[3]int` 和 `[4]int` 是不同的类型。
- **声明数组：**
	- ```go
	  [数组中的元素数量]数组保存元素的类型
	  
	  var a [3]int       // 声明一个保存 3 个 int 的数组
	  a := [3]int
	  ```
- **初始化数组：**
	- 如果知道初始值，可以使用数组字面量进行初始化：
	- ```go
	  var a [3]int = [3]int{1, 2, 3}
	  var a = [3]int{1, 2, 3}        // 类型由值推断
	  a := [3]string{"Go", "Rust", "Python"}
	  arr := [...]float64{3.14, 2.71, 1.41} // 使用 `...` 自动推导长度
	  ```
- `len(v)` 函数用于获取长度，其中 `v` 可为数组、切片、字符串、map 或 channel：
	- 数组：返回固定长度
	- 切片：返回当前元素个数
	- 字符串：返回字节数（不是字符数）
	- map：返回键值对数量
	- channel：返回缓冲区中的元素数量
