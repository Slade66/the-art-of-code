## 算术运算符（Arithmetic Operators）
	- | **运算符** | **描述** | **示例** | **注意事项** |
	  | ---- | ---- | ---- |
	  | `+` | 相加 | `A + B` | 也用于**字符串拼接** (`"He" + "llo"`) |
	  | `-` | 相减 | `A - B` |  |
	  | `*` | 相乘 | `A * B` |  |
	  | `/` | 相除 | `A / B` | 两个整数相除，结果会**截断小数** (`5/2` 结果是 `2`) |
	  | `%` | 求余 | `A % B` | 只能用于整数 |
	  | `++` | 自增 | `A++` | **重要：** 只能作为**独立语句**，不能放在表达式中 |
	  | `--` | 自减 | `A--` | 同上 |
	- **Go 语言中的 `++` 和 `--`**
		- ```go
		  i := 1
		  i++          // ✅ 合法
		  // j := i++  // ❌ 编译错误：syntax error
		  // ++i       // ❌ 编译错误：syntax error
		  ```
		- **只有后置自增：**不支持 `++a`，只有 `a++`。
		- **不返回值：**`a++` 是一个语句（Statement），不是表达式（Expression），它不产生值，因此不能赋值给变量。
- ## 关系运算符（Relational Operators）
	- 用于比较两个值，结果永远是 `bool` 类型（`true` 或 `false`）。
	- | **运算符** | **描述** | **示例** |
	  | ---- | ---- | ---- |
	  | `==` | 等于 | `A == B` |
	  | `!=` | 不等于 | `A != B` |
	  | `>` | 大于 | `A > B` |
	  | `<` | 小于 | `A < B` |
	  | `>=` | 大于等于 | `A >= B` |
	  | `<=` | 小于等于 | `A <= B` |
	- **注意**：Go 是强类型语言，不同类型的变量（例如 `int32` 和 `int64`）不能直接比较，必须先进行显式类型转换。
- ## 逻辑运算符（Logical Operators）
	- ```
	  +------+----------+---------+----------------------------------+
	  | 运算符 | 描述     | 示例    | 特性                             |
	  +------+----------+---------+----------------------------------+
	  | &&   | 逻辑与    | A && B  | 如果 A 为 false，不再计算 B      |
	  | ||   | 逻辑或    | A || B  | 如果 A 为 true，不再计算 B       |
	  | !    | 逻辑非    | !A      | 取反                             |
	  +------+----------+---------+----------------------------------+
	  
	  ```
- ## 位运算符（Bitwise Operators）
	- ```
	  +------+----------------------+-----------+-----------------------------------------------+
	  | 运算符 | 描述                 | 示例      | 说明                                          |
	  +------+----------------------+-----------+-----------------------------------------------+
	  |  &   | 按位与               | A & B     | 逐位比较，只有两位都为1时结果为1。            |
	  |      |                      |           | 示例： A = 1101                                  |
	  |      |                      |           |        B = 1011                                  |
	  |      |                      |           | A & B = 1001 (只有第3和第0位都为1) => 1001 (9) |
	  +------+----------------------+-----------+-----------------------------------------------+
	  |  |   | 按位或               | A | B     | 逐位比较，只要有一位为1结果就为1。            |
	  |      |                      |           | 示例： A = 1101                                  |
	  |      |                      |           |       B = 1011                                  |
	  |      |                      |           | A | B = 1111 (任一位为1都保留) => 1111 (15)      |
	  +------+----------------------+-----------+-----------------------------------------------+
	  |  ^   | 按位异或             | A ^ B     | 逐位比较，位不同则为1，相同则为0。              |
	  |      |                      |           | 示例： A = 1101                                  |
	  |      |                      |           |       B = 1011                                  |
	  |      |                      |           | A ^ B = 0110 (不同位变1) => 0110 (6)           |
	  +------+----------------------+-----------+-----------------------------------------------+
	  | <<   | 左移                 | A << 2    | 各二进位向左移若干位：高位丢弃，低位补0。      |
	  |      |                      |           | 示例： A = 0000 1101 (8位显示)                 |
	  |      |                      |           | A << 2 = 0011 0100 => 110100 (即原数乘以4)     |
	  +------+----------------------+-----------+-----------------------------------------------+
	  | >>   | 右移                 | A >> 2    | 各二进位向右移若干位：低位丢弃，高位补符号/0。 |
	  |      |                      |           | 示例（无符号或逻辑右移）：                      |
	  |      |                      |           | A = 0000 1101                                   |
	  |      |                      |           | A >> 2 = 0000 0011 => 11 (即原数除以4，取整)  |
	  +------+----------------------+-----------+-----------------------------------------------+
	  | &^   | 按位清空 (AND NOT)   | A &^ B    | Go 特有：将 A 中对应 B 为1的位清零（把B当“清除掩码”）。|
	  |      |                      |           | 示例： A = 1101                                  |
	  |      |                      |           |       B = 1011                                  |
	  |      |                      |           | A &^ B = 0100 (把 A 中 B 为1的位置清成0) => 0100 (4) |
	  +------+----------------------+-----------+-----------------------------------------------+
	  
	  ```
- ## 特殊运算符：指针与通道
	- **指针运算符（Pointer Operator）：**
		- `&`（取地址）：获取变量在内存中的地址。
		- `*`（解引用）：获取指针指向的内存地址中的值。
		- ```go
		  a := 10
		  ptr := &a  // ptr 存储了 a 的地址
		  val := *ptr // val 获取了 ptr 指向的值 (10)
		  ```
	- **通道运算符（Channel Operator）：**
		- `<-` ：用于发送或接收数据。箭头的方向指示了数据的流向。
		- ```go
		  ch := make(chan int)
		  ch <- 10    // 发送：把 10 发送到通道 ch
		  x := <-ch   // 接收：从通道 ch 取出数据赋值给 x
		  ```
- ## 运算符的优先级
	- 位移运算符 `<<` 的优先级高于加法 `+`。
- ## 注意
	- 没有三元运算符（`? :`），请使用 `if-else` 替代。
