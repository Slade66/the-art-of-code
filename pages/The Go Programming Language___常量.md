- 在 Go 语言中，`const` 用于声明在编译期间值已确定且不可更改的常量。
- **不可变性：**一旦声明，常量的值就不能被修改。
- **常量是高精度值：**
  collapsed:: true
	- **高精度运算：**常量是无限精度的，只受限于可用的内存，而不是像普通浮点数那样有固定的精度限制。这意味着可以进行非常大的数字运算而不会丢失精度。
	- **无类型特性：**常量本身不具备类型，其类型是在被赋值给变量、传递给函数参数或进行显式类型转换时才被确定的。
	- **类型安全检查：**如果一个常量的值超出了其目标类型的表示范围，或者无法用该类型精确表示，编译器会直接报错。
	- **示例：**
		- ```go
		  const x = 1.0/3.0
		  var y float32 = x * 3
		  println(y)
		  ```
		- 常量表达式 `1.0/3.0` 是无类型的且精度极高，乘以 3 后精确还原为 1.0，赋值给 y 时就是 1.0。
- **声明常量的语法：**
  collapsed:: true
	- **注意：**
		- 常量必须在声明时赋值，不能在之后再赋。
		- 不支持使用短变量声明（`:=`）的方式，必须使用 `const` 关键字。
	- **单个常量声明：**
		- **语法：**`const 常量名 类型 = 值`
		- ```go
		  const Pi float64 = 3.14159
		  const Pi = 3.14 // 类型可以省略，Go 会根据赋的值自动推断类型
		  ```
	- **批量声明常量：**
		- ```go
		  const (
		      A = 1
		      B = 2
		      C = 3
		  )
		  
		  const (
		      A = 1
		      B        // B = 1
		      C = 3
		  )
		  ```
		- 在 `const` 括号块中，如果一行留空（没有写类型和表达式），Go 编译器会自动照抄上一行的类型和表达式。
	- **使用 `iota` 生成递增值：**
		- `iota` 是 Go 语言中用于在 `const` 声明块内自动递增的计数器，从 0 开始。
		- **起始值**：每当进入一个新的 `const` 块时，`iota` 会被重置为 0。
		- **递增规则**：在同一个 `const` 块中，每出现一行新的常量声明，`iota` 的值就会加 1；空行或注释不会影响递增。
		- **类型**：`iota` 是一个无类型的整数常量。
		- ```go
		  const (
		      A = iota  // iota = 0, A = 0
		      B         // iota = 1, B = 1 (B = iota 表达式被隐式重复)
		      C         // iota = 2, C = 2 (C = iota 表达式被隐式重复)
		  )
		  
		  const (
		      D = iota  // iota 重新设置为 0, D = 0
		      E         // iota = 1, E = 1
		  )
		  ```
		- **使用空白标识符 `_` 来跳过 `iota` 枚举中的一个或多个值。**
			- `iota` 在每个 `const` 声明块中都会递增，即使被赋值给 `_` 也会递增。
			- ```go
			  package main
			  
			  import "fmt"
			  
			  const (
			  	// iota = 0
			  	a = iota // a = 0
			  
			  	// iota = 1，将值分配给空白标识符，从而跳过
			  	_ = iota
			  
			  	// iota = 2
			  	c = iota // c = 2
			  
			  	// iota = 3
			  	d = iota // d = 3 (作为额外示例)
			  )
			  
			  func main() {
			  	fmt.Printf("a = %d\n", a)
			  	fmt.Printf("c = %d\n", c)
			  	fmt.Printf("d = %d\n", d)
			  }
			  ```
- **注意：**
	- Go 的 `const` 只能用于数字、字符、字符串、布尔，不能用于结构体、切片、指针等复合类型。
	- **未使用的常量不报错，与变量严格的“定义必使用”规则不同。**
