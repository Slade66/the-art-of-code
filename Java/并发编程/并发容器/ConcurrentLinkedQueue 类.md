`ConcurrentLinkedQueue` 是 Java 并发包（java.util.concurrent）提供的一个线程安全的、非阻塞的、基于链表的队列。在队列为空时，执行出队操作会立刻返回，而不会像阻塞队列那样使线程阻塞等待新元素的到来。

**常用方法**

- `offer(E e)`: 将指定元素插入队列尾部。由于该队列是非阻塞且无界的，该方法通常会立即返回 `true` 表示成功；在极少数情况下，如资源受限，可能返回 `false`。
- `poll()`: 移除并返回队列头部的元素；如果队列为空，则返回 `null`。
- `peek()`: 返回队列头部的元素，但不移除它；如果队列为空，则返回 `null`。
- `isEmpty()`: 判断队列是否为空，若无元素则返回 `true`。
- `size()`: 返回队列中的元素数量。
- `contains(Object o)`: 如果队列包含指定的元素，则返回 `true`。
